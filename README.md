## 🎥 BarberBox
Um sistema de agendamento online para barbearias, focado em arquitetura limpa, organização estrutural e experiência do usuário.

### Link:
- https://barberbox-eight.vercel.app/home

### 🛠️ Tech Stack:
- React
- Next.js
- TypeScript
- Tailwind
- PostgreSQL
- JWT
- Git
